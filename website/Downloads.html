EasyMock Class Extension

This extension used to allow generating Mock Objects for classes. It is now deprecated since EasyMock code package now perform class mocking. It has been contributed by Joel Shellman, Henri Tremblay and Chad Woolley. Henri Tremblay maintains this extension.
